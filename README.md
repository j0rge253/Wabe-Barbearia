# Wave Barbearia
Wave Barbearia 💈
Sistema Full Stack para gestão de agendamentos.

Diferencial: Implementação de fluxo de agendamento público (sem necessidade de login para o cliente), aumentando a conversão de serviços.

Painel Administrativo: Gestão de horários, serviços e faturamento em tempo real para o barbeiro.

Tech: NestJS (Backend), React + Tailwind (Frontend), Prisma + PostgreSQL (Banco de Dados).
